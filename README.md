# SOLID-design-patterns

Лабораторная работа №06: Паттерны проектирования

## Описание

Реализация редактора подземелья для RPG игры Balagur Fate 3. Программа моделирует базовые локации с расстановкой NPC (Медведь, Выпь, Выхухоль).

## Функционал

- Добавление NPC по координатам (0<=x,y<=500)
- Сохранение и загрузка NPC из файла
- Печать списка NPC
- Боевой режим с указанием дальности: NPC сражаются каждый с каждым

## Правила совместимости

- Медведь ест всех кроме медведей
- Выпь никого не обижает
- Выхухоль убивает медведей

## Паттерны SOLID

- Factory Method для создания NPC
- Visitor для проведения сражения
- Observer для логирования убийств (консоль и файл log.txt)

## Сборка и запуск

```bash
mkdir build
cd build
cmake ..
make
./SOLID_design_patterns_exe
```

## Тесты

```bash
./tests
```
